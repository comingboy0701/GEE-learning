# 第2节 图像的基本操作


## 1 图像的卷积


### 1.1 一阶导数
相邻元素相减: 
$f(x)' = f(x+1)-f(x)$


### 1.2 二阶导数

$f(x,y)'' = f(x+1,y)-2f(x,y)+f(x-1,y)$

$f(x,y)'' = f(x,y+1)-2f(x,y)+f(x,y-1)$

即，拉普拉斯（Laplacian）： 

0  1   0  

1  -4  1 

0   1  0  


### 1.3 高斯核

- $f(x) = \frac{1}{\sigma \sqrt{2 \pi}} \exp \left(-\frac{(x-\mu)^{2}}{2 \sigma^{2}}\right)$

- 加速运算: $f(x) = G(x)*G(y)$


### 1.4 其它核的操作

- 图像锐化: Laplacian
- 边缘检测: Solel
- 图像模糊：Meidan/Gaussian


## 2 图像的特征点


### 1.2 好的特征点

 角点的信息量比较大
 
 - 平移：敏感
 - 旋转：不敏感
 - 缩放（尺度）：敏感


### 1.3 SIFT


- 边缘检测
- 极值点
- 插值
- 消除边缘
- 梯度幅值
- 关键点描述
